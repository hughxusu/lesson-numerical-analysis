# 数值分析

**本站网址**  https://hughxusu.github.io/lesson-numerical-analysis/#/

**[代码仓库](https://github.com/hughxusu/lesson-numerical-analysis)**

## 课程目标

* 学习数值分析的基本概念
* 学习数值分析的方法
  1. 插值法
  2. 函数逼近与曲线拟合
  3. 数值积分和数值微分
  4. 解线性代数方程组的直接法
  5. 解线性代数方程组的迭代法
  6. 非线性方程、方程组的数值解法
  7. 矩阵特征值计算
  8. 常微分方程初值问题数值解法
